 Book Sales & Ratings SQL Analysis

Project Overview
This project analyzes book sales and ratings data using SQL to uncover insights about author performance, genre trends, and revenue generation.
The analysis focuses on identifying top-selling books, high-performing authors, and hidden gems based on ratings and sales.

Tools & Skills Used
* SQL (MySQL)
* Joins, Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Data Analysis & Business Insights

 Dataset Information
* Book details (name, author, genre, publisher)
* Ratings (average rating, author rating)
* Sales metrics (gross sales, units sold, revenue)
* Pricing information

Key Analysis Performed
🔹 Sales Analysis
 * Top-selling books and authors
 * Revenue by publisher
 * Units sold analysis

🔹 Ratings Analysis
 * Average rating by genre
 * High-rated books
 * Author performance

🔹 Advanced Insights
 * Hidden gems (high rating, low sales)
 * Profit margin per book
 * Price vs rating relationship

 Key Insights
* Certain authors consistently generate high revenue
* High-rated books do not always have high sales
* Some genres dominate both ratings and revenue
* Hidden gems exist with strong ratings but low visibility

Project Structure
1. dataset.csv
2. schema.sql
3. queries.sql
4. insights.md
5. README.md

This analysis helps:
* Publishers identify profitable books
* Businesses understand customer preferences
* Improve marketing for low-selling high-rated books
